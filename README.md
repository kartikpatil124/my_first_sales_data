# 📊 Sales Data Analysis with Python

This project performs **sales data analysis using Python and CSV files**.
The purpose of this project is to practice **Python file handling, data processing, and basic data analysis**.

The dataset contains information about **orders, customers, products, categories, quantities, prices, and order dates**.

Using Python, the project answers several real-world business questions from the dataset.

---

# 🚀 Project Features

This project performs the following sales analysis tasks:

✔ Count total number of orders
✔ Calculate total quantity of items sold
✔ Calculate total revenue generated
✔ Calculate revenue generated by each category
✔ Find the product with the highest quantity sold
✔ Find the customer who spent the most money
✔ Calculate revenue for each date
✔ Calculate total revenue between two given dates
✔ Export all **Electronics category orders** into a new CSV file

These are common **data analysis tasks used in real business scenarios**.

# 📁 Dataset Columns

The dataset contains the following columns:

| Column   | Description               |
| -------- | ------------------------- |
| OrderID  | Unique order ID           |
| Date     | Date of the order         |
| Customer | Customer name             |
| Product  | Product purchased         |
| Category | Product category          |
| Quantity | Number of items purchased |
| Price    | Price per item            |

---

# 🧠 Example Revenue Calculation

Revenue is calculated using:

```
Revenue = Quantity × Price
```

Example:

```
2 × 1999 = 3998
```

This means if a customer buys **2 items priced at 1999**, the revenue from that order is **3998**.
